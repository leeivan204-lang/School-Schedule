# 特教班行事曆 (School Schedule) 使用說明書

歡迎使用「特教班行事曆」產生器！本軟體專為特教班設計，能協助您快速建立、管理並列印美觀的學期行事曆。

## 🚀 快速入門

### 1. 開啟程式
請依照開發者提供的網址或本機啟動方式開啟網頁。
- 若為主機檔案，請執行啟動腳本後，瀏覽器將自動開啟。

### 2. 介面總覽
畫面主要分為三個區域：
- **頂部標題區**：設定學年與學期。
- **設定與輸入區**：設定開學日、切換檢視模式、新增事項與備註。
- **行事曆預覽區**：即時顯示目前的行事曆排版。

---

## 🛠️ 功能詳細說明

### 1. 設定學期資訊
在畫面最上方的標題區，您可以直接修改：
- **年份**：例如 `114`。
- **學期**：例如 `1` 或 `2`。
*系統會自動記憶您最後輸入的數值。*

### 2. 設定學期開始日期
在「設定與輸入區」找到 **「學期開始日期」**。
- 請選擇該學期 **第一週的星期日**。
- 更改後，下方的行事曆週次與日期會自動重新計算。

### 3. 新增行事曆事項 (重要事項)
在「新增事項」區塊依照步驟輸入：
1. **開始日期**：必填。
2. **結束日期**：選填。若為單日活動則留空；若為連續多日活動（如段考週），請選擇結束日。
3. **事項內容**：輸入活動名稱，例如「第一次定期評量」。
4. **僅限教師**：勾選此選項後，該事項只會在「教師版」顯示，「家長版」會自動隱藏。
5. 點擊 **「新增事項」** 按鈕。

> **💡 智慧變色提示**：
> 系統會根據關鍵字自動套用顏色：
> - 🔴 **段考**：紅色標示
> - 🟢 **校外教學**：綠色標示
> - 🟠 **慶生會**：橘色標示
> - 🔵 **節日/補假/放假**：藍色標示

### 4. 新增月份備註
在「新增月份備註」區塊：
1. **月份**：選擇要加入備註的月份。
2. **備註內容**：輸入文字，例如「孔子誕辰紀念日」。
3. 點擊 **「新增備註」** 按鈕。
*備註會顯示在該月份右側的「備註」欄位中。*

### 5. 刪除資料
- **刪除事項**：在行事曆表格中，**雙擊 (Double Click)** 該事項的色塊或文字，確認後即可刪除。
- **刪除備註**：在右側備註欄位中，**雙擊** 該條備註文字，確認後即可刪除。

### 6. 切換檢視模式 (教師版 / 家長版)
位於畫面中間的切換開關：
- **教師版**：顯示「所有」事項（包含標記為僅限教師的活動）。適合列印給校內同仁使用。
- **家長版**：隱藏「僅限教師」的事項。適合列印發放給家長。

## 💾 儲存與輸出

### 自動儲存
本系統具備 **自動儲存** 功能。您輸入的所有資料（日期、事項、備註）都會儲存在您的瀏覽器中。下次開啟同一瀏覽器時，資料會自動載入。

### 匯出 PDF / 列印
點擊 **「匯出 PDF / 列印」** 按鈕：
1. 瀏覽器會跳出列印視窗。
2. 目的地請選擇 **「另存為 PDF」** 即可儲存檔案，或選擇印表機直接列印。
3. 建議設定：
   - 版面配置：**橫向**
   - 邊界：**無** 或 **最小** (以確保版面完整)
   - 勾選 **「背景圖案」** (Background graphics)，以確保顏色區塊正確顯示。

## ❓ 常見問題 Q&A

**Q: 為什麼跨月份的週次顯示怪怪的？**
A: 系統以「週」為單位，會將該週歸類在佔比較多的月份。

**Q: 如何修改已經建立的事項？**
A: 目前不支援直接修改。請直接 **雙擊刪除**舊事項，再重新新增一筆正確的即可。

**Q: 換了電腦資料還在嗎？**
A: 資料是儲存在「目前的瀏覽器」中。若更換電腦或瀏覽器，資料不會同步，需要重新輸入。
